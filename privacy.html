Privacy Policy

김홍영화학교육연구소는 수험생 개인의 학습 성과 분석을 위해 최소한의 개인정보만을 안전하게 수집하고 처리합니다. 본 사이트는 Google OAuth 로그인을 통해 개인정보를 인증받으며, 아래 내용을 따릅니다.

1. 수집 항목 (Information We Collect)

  • Google OAuth를 통해 인증된 정보: 이름, 이메일 주소, 수험번호

2. 수집 및 이용 목적 (Purpose of Use)

  • 개별 학습 성과 리포트 자동 생성
  • 수험생 성적 분석 및 맞춤형 학습 피드백 제공

3. 보유 및 이용 기간 (Data Retention)

수집된 데이터는 리포트 생성 후 최대 7일 이내에 자동 삭제되며, 그 외의 목적에는 절대 사용되지 않습니다.

4. 제3자 제공 (Third Party Disclosure)

개인정보는 외부 기관에 제공되지 않으며, 단 다음의 경우 예외로 합니다:

  • 법령에 따라 요청이 있을 경우
  • 사용자가 명시적으로 동의한 경우

5. 데이터 보호 조치 (Security Measures)

  • OAuth 인증을 통해 수집된 개인정보는 SSL 암호화된 연결을 통해 안전하게 전송됩니다.
  • 데이터는 Google Cloud Platform 내 보안 서버에 저장되며, 인증된 관리자만 접근할 수 있습니다.
  • 사용자의 개인정보는 수험 결과 리포트 생성 목적 외에는 절대 사용되지 않으며, 7일 이내 자동 삭제됩니다.
  • 본 서비스는 Google API Services User Data Policy에 따라 민감한 사용자 정보를 안전하게 보호합니다.

6. 개인정보 파기 방법 (Data Deletion)

보관 기간이 경과된 개인정보는 지체 없이 안전한 방식으로 자동 삭제되며, 물리적/논리적 방식으로 완전하게 파기됩니다.

Privacy Policy (English)

Kim Hong Young Educational Institute values your privacy and collects only the minimum amount of personal data through Google OAuth for academic purposes.

1. Information We Collect

  • Name
  • Email address
  • Student ID number

2. Purpose of Use

Collected data is used solely to generate personalized academic score reports and provide performance feedback for students.

3. Data Retention

User data is automatically deleted within 7 days after generating the report. It is never reused for any other purpose.

4. Third Party Sharing

We never share any personal data with third parties unless legally required or explicitly consented by the user.

5. Security Measures

  • All data is transmitted securely via SSL-encrypted connections.
  • Stored securely on Google Cloud with access restricted to verified administrators only.
  • All data is removed after 7 days using secure deletion methods.
  • We comply with the Google API Services User Data Policy.
