Privacy Policy

김홍영화학교육연구소는 수험생 개인의 학습 성과 분석을 위해 최소한의 개인정보만을 안전하게 수집하고 처리합니다. 본 사이트는 Google OAuth 로그인을 통해 개인정보를 인증받으며, 아래 내용을 따릅니다.

1. 수집 항목 (Information We Collect)

  • Google OAuth를 통해 인증된 정보: 이름, 이메일 주소, 수험번호

2. 수집 및 이용 목적 (Purpose of Use)

  • 개별 학습 성과 리포트 자동 생성
  • 수험생 성적 분석 및 맞춤형 학습 피드백 제공

3. 보유 및 이용 기간 (Data Retention)

수집된 데이터는 리포트 생성 후 최대 7일 이내에 자동 삭제되며, 그 외의 목적에는 절대 사용되지 않습니다.

4. 제3자 제공 (Third Party Disclosure)

개인정보는 외부 기관에 제공되지 않으며, 단 다음의 경우 예외로 합니다:

  • 법령에 따라 요청이 있을 경우
  • 사용자가 명시적으로 동의한 경우

5. 데이터 보호 조치 (Security Measures)

  • 접근 제어 및 암호화를 통한 보안 서버 저장
  • Google API Services User Data Policy 준수
  • 관리자 외 접근 불가능한 스토리지로 제한

6. 개인정보 파기 방법 (Data Deletion)

보관 기간 경과 후 안전한 방식으로 자동 삭제되며, 물리적/논리적 파기를 병행합니다.

Privacy Policy (English)

We collect user data through Google OAuth to generate personalized academic score reports for students.

1. Information We Collect

  • Name
  • Email address
  • Student ID number

2. Purpose of Use

The collected data is used only to generate personalized exam result reports and to provide performance feedback. No data is used for advertising or shared with third parties.

3. Data Retention

User data is stored temporarily and deleted within 7 days after the generation of the academic report.

4. Security

We use encrypted storage, restricted access, and comply with the Google API Services User Data Policy.

5. Third Party Sharing

We never share user data unless legally required or with explicit user consent.
